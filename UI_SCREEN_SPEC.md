# UI Screen Specification
寮管理アプリ 画面仕様書

本アプリはスマートフォンでの利用を前提とする。
Streamlit を利用して画面を構築する。

画面は Streamlit の pages 機能を利用して構成する。

---

# 1 ダッシュボード

ファイル

dashboard.py

目的

ログイン後に表示されるトップページ。

表示内容

- 今日の掃除当番
- 未払い集金
- 最新のお知らせ
- 最新アルバム投稿
- 最近の掲示板投稿

UI

カード形式で表示

例

[掃除当番]
キッチン
担当：山田

[未払い集金]
寮費 5000円

[お知らせ]
今週の掃除スケジュール
