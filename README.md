# 寮管理アプリ (Dormitory Management App)

## 概要

社員寮のメンバーが情報共有できるグループウェアアプリ。

寮メンバー（約20人）を対象とし、スマートフォンから利用することを前提とする。

本アプリは以下の目的で利用する。

- 寮メンバーの管理
- 掃除当番の管理
- 集金管理
- 物品管理
- お知らせ共有
- 掲示板コミュニケーション
- アルバム共有
- 寮規約の共有
